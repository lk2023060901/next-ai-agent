| 个人 | Premium | ¥199/月 | ¥159/月 | 专业用户 / 自由职业者 |

#### 2.2.1 标签切换器 (Tabs)

**位置**: 标题下方, 居中

**样式**: Pill/Segment 风格 (非下划线), 与月付/年付 Toggle 上下分层

```
┌──────────────────────────────────────┐
│  [ 个人 ]  │  [ 企业 ]  │  [ 定制 ]  │
└──────────────────────────────────────┘
```

- 容器: 背景 Surface, 圆角 radius-full, 内边距 4px, 边框 1px Border
- 每个标签: 内边距 8px 20px, 圆角 radius-full
- 默认态: 文字 Text-Secondary, 背景 transparent
- 选中态: 文字 #FFFFFF, 背景 Primary-500, font-weight 600, 阴影 shadow-sm
- Hover (未选中): 文字 Text-Primary, 背景 Surface-2
- 切换动画: 选中背景滑动, duration-normal, easing-default
- 标签与月付/年付 Toggle 之间间距: space-4

**月付/年付 Toggle**:
- 位于标签下方, 居中
- 样式: Switch + 文字标签
- "月付" (Body-sm) [Toggle] "年付" (Body-sm) + "省20%" Badge (Success-50 背景, Success-600 文字, Tiny, radius-full, 内边距 2px 8px)
- 切换时卡片价格数字带过渡动画: 数字向上淡出 → 新数字向下淡入, duration-fast

#### 2.3.4 升级/降级规则

| 操作 | 生效时间 | 计费方式 |

## 3 用量超额处理

### 3.1 告警阈值

| 阈值 | 动作 |
|------|------|
| 80% | 邮件通知 + 站内通知 (Warning) |
| 95% | 邮件通知 + 站内通知 + Dashboard 横幅 (Danger) |
| 100% | Agent 服务暂停, 跳转升级页面 |

### 3.2 超额横幅

位于主内容区顶部, 全宽:

**Warning (80%)**:
```
┌─ ⚠ ──────────────────────────────────────────────────────┐
│ Token 使用量已达 80% (1.6M / 2M), 建议升级方案             │
│                                      [查看用量] [升级]     │
└──────────────────────────────────────────────────────────┘
```
- 背景: Warning-50
- 边框: 1px Warning-200
- 图标: AlertTriangle (Warning, 20px)
- 可关闭 (X 按钮)

**Danger (95%+)**:
- 背景: Danger-50
- 边框: 1px Danger-200
- 不可关闭

### 3.3 超额后升级引导

Agent 服务暂停后, 对话输入区替换为:

```
│  当前方案: Pro (2M Token/月)                               │
│  (Body-sm, Text-Secondary, 居中)                          │
│                                                           │
│  [升级方案](Primary md)  [查看用量详情](Ghost md)           │
│                                                           │
└──────────────────────────────────────────────────────────┘
```

- 背景: Danger-50
- 圆角: radius-lg
- 内边距: space-6
