### 1.3 统计卡片

#### 1.3.1 通用结构

```
┌──────────────────────────────┐
│ [图标 24px, 对应色]   [趋势]  │
│                               │
│ 12  (H1, Text-Primary)       │
│ 活跃 Agent (Caption, Text-Sec)│
│                               │
│ [迷你图表 高40px]              │
└──────────────────────────────┘
```

- 内边距: space-5
- 圆角: radius-lg
- 背景: Surface
- 边框: 1px Border
- 趋势标记: 右上角, ↑12% (Success, Tiny) 或 ↓5% (Danger, Tiny)
- 迷你图表: Sparkline, 无坐标轴, 7 天数据点

#### 1.3.2 各卡片数据

| 卡片 | 图标 | 图标色 | 数值 | 单位 | 图表类型 |
|------|------|--------|------|------|---------|
| 活跃 Agent | Bot | Purple | 整数 | 个 | 折线 |
| 今日会话 | MessageSquare | Blue | 整数 | 次 | 柱状 |
| Token 用量 | Zap | Amber | K/M 缩写 | tokens | 面积 |
| 完成任务 | CheckSquare | Green | 整数 | 个 | 折线 |

### 1.4 消息量趋势图

- 类型: 面积折线图 (Area Chart)
- 高度: 280px
- X 轴: 日期 (最近 7 天), 刻度间距均匀
- Y 轴: 消息数, 自动计算刻度
- 数据系列:
  - 入站消息 (Primary-500, 填充 Primary-50)
  - 出站消息 (Success-500, 填充 Success-50)
- 图例: 右上角, 水平排列
- Tooltip: hover 显示 (日期 + 入站数 + 出站数)
- 网格线: 水平, Border 色, 虚线

### 1.5 Agent 工作负载

- 类型: 环形图 (Donut Chart)
- 高度: 280px
- 中心: 总任务数 (H2) + "任务" (Caption)
- 每个 Agent 占一段, 使用 Agent 角色色
- 图例: 下方, 网格排列 (2 列)
  - 每项: 色块 (12x12, radius 2px) + Agent 名 (Body-sm) + 数量 (Body-sm, 600)

### 1.6 最近活动

- 见工作区首页 2.3.2 最近活动, 样式一致
- 额外: 筛选下拉 (全部 / 仅 Agent 活动 / 仅成员活动)

### 1.7 快捷操作

垂直排列的操作按钮卡片 (间距 space-3):

```
┌───────────────────────────────────┐
│ [Plus 图标]  新建对话              │
│              与 Agent 开始交谈      │
└───────────────────────────────────┘
┌───────────────────────────────────┐
│ [Upload 图标]  上传知识             │
│                添加文档到知识库      │
└───────────────────────────────────┘
┌───────────────────────────────────┐
│ [Settings 图标]  管理 Agent        │
│                  配置和调整 Agent    │
└───────────────────────────────────┘
┌───────────────────────────────────┐
│ [Users 图标]  邀请成员              │
│               添加团队成员          │
└───────────────────────────────────┘
```

- 每项: 内边距 space-3, 圆角 radius-md, 背景 Background, 边框 1px Border
- 左侧图标: 20px, Primary-400
- 标题: Body, font-weight 500
- 描述: Caption, Text-Tertiary
- hover: 背景 Primary-50, border Primary-200
- 点击: 跳转到对应页面

## 2 用量统计页面

### 2.1 页面路由

**路由**: `/org/[slug]/usage`

### 2.2 页面布局

**顶部筛选器**:
- 时间范围: DateRangePicker (默认最近 30 天)
- 工作区: Select (全部 / 指定工作区)
- Agent: Select (全部 / 指定 Agent)
- "导出 CSV" 按钮 (Secondary, sm, 右对齐)

### 2.3 用量概览 (4 列)

| 指标 | 数值格式 | 备注 |
|------|---------|------|
| 总 Token 消耗 | 数字 + 单位 (K/M) | 含 input + output |
| API 调用次数 | 整数 | 所有 Provider 总计 |
| 平均响应时间 | ms | P50 |
| 预估费用 | ¥ / $ | 基于 token 单价 |

### 2.4 用量详情图表

#### 2.4.1 Token 消耗趋势

- 类型: 堆叠柱状图
- X 轴: 日期
- Y 轴: Token 数 (K)
- 系列: Input Tokens (Primary-300) + Output Tokens (Primary-600)
- 高度: 300px

#### 2.4.2 Provider 分布

- 类型: 饼图
- 按 Provider 分: Anthropic / OpenAI / Google / 其他
- 颜色: 各 Provider 品牌色

#### 2.4.3 Agent 用量排行

- 类型: 水平柱状图
- Y 轴: Agent 名称 (按用量排序)
- X 轴: Token 消耗
- 颜色: Agent 角色色
- 高度: 按 Agent 数量自适应 (每条 40px)

### 2.5 明细表格

| 列 | 宽度 | 内容 |
|-----|------|------|
| 时间 | 15% | Caption |
| Agent | 15% | 角色色圆点 + 名称 |
| Provider | 10% | Logo + 名称 |
| 模型 | 15% | 模型名 |
| Input Tokens | 10% | 数字, 右对齐 |
| Output Tokens | 10% | 数字, 右对齐 |
| 耗时 (ms) | 10% | 数字, 右对齐 |
| 费用 | 10% | ¥/$ 金额 |
| 状态 | 5% | ✓ / ✗ |

- 支持按列排序 (点击表头)
- 分页: 每页 100 条
